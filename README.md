
# 说明
某盟的ADS可以通过黑洞路由，实现IP地址封堵的功能（将封堵IP指向黑洞路由），该类设备提供API接口，可以批量提交恶意IP进行封堵，所以利用这个API写了个一调用的接口。
# 使用前提
需要在ADS上配置号第三方接口，配置方式参考某盟设备，本代码不提供接口文档。

# 使用环境
Python3.x
```
pip install -r requirements.txt
```
# 使用方法
1、修改sqlite3.db的配置文件，添加ads接口
2、添加需要封堵的IP到excel模板中
3、执行deny.py文件

代码中存在一些恶趣味，可自行修改。
